## Suggested build order
Start small and layer complexity progressively:

1. Matrix class — heap-allocated 2D array, operator overloading (+, *, []), copy vs. move constructors. This alone teaches you pointers, RAII, and the Rule of Five.
1. Autograd / backprop — teaches you graph structures, shared_ptr vs unique_ptr, and lifetime management.
1. Layer abstraction — a base Layer class with forward() and backward() as pure virtual methods. Teaches inheritance and vtables.
1. Templated types — make Matrix<T> work for any numeric type. Teaches templates and specialization.
1. Optimizer — SGD, then Adam. Teaches functors and std::function.
1. Train on something real — XOR first, then MNIST.
